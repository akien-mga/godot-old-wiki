#  Position2D  
####**Inherits:** [Node2D](class_node2d)
####**Category:** Core

###  Brief Description  
Generic 2D Position hint for editing.

###  Description  
Generic 2D Position hint for editing. It's just like a plain [Node2D](class_node2d) but displays as a cross in the 2D-Editor at all times.
