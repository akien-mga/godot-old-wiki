#  PinJoint2D  
####**Inherits:** [Joint2D](class_joint2d)
####**Category:** Core

###  Brief Description  
Pin Joint for 2D Shapes.

###  Description  
Pin Joint for 2D Rigid Bodies. It pins 2 bodies (rigid or static) together, or a single body to a fixed position in space.
